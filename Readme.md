###### Hero Stats

There are two types of stats, Attributes and Stats

Attributes defines your stats:

STR {
Increase ATK
Increase HP
}

DEX {
Increase ACC
Increase DEF
}

AGI {
Increase SPD
}

VIT {
Increase HP
Increase DEF
}

INT {
Increase MAG
Increase SP
}

}

SP {
Skill Points (Mana)
}

ATK {
Melee Damage
}

MAG {
Magic Damage
}

DEF {
Defense, damage mitigate
}

ACC {
Accuracy, chance to hit
}

SPD {
Speed, Atack Speed/Chance to Dodge
}
